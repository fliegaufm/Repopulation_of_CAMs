# Repopulation_of_CAMs
Code used to analyze data for "Distinct origins and niches determine the cellular responsiveness of CNS macrophages after repopulation"

Its purpose is to provide explanation on how to reproduce key findings of this publication.

Users need to install R programming language. It can be downloaded from: https://cran.r-project.org/

For more information, issues or discussion of data or analysis, please reach out to maximilian.fliegauf@uniklinik-freiburg.de

All data can be accessed through the following GEO accession codes: 

GSE318650	scRNA sequencing data of cells from leptomeninges, cortex and dura mater from CTL, 5d and 8 weeks after macrophage deletion (scRNAseq)

GSE295335	RNA sequencing data of leptomeningeal/perivascular CAMs 8 weeks after repopulation (scRNAseq)

GSE294773	RNA sequencing data of leptomeningeal/perivascular CAMs 8 weeks after repopulation

GSE318386	scATAC sequencing data of subdural, dural CAMs and MG 8 weeks after repopulation

GSE295334	RNA sequencing data of leptomeningeal/perivascular CAMs and MG following LPS stimulation 8 weeks after repopulation

GSE318071	RNA sequencing data of dural CAMs 8 weeks after repopulation stimulated with LPS

GSE294912	RNA sequencing data of whole tissue or CAMs isolated from WT mice after ischemic stroke

GSE318159	RNA sequencing data of fate-mapped leptomeningeal/perivascular CAMs 8 weeks after repopulation and 1 day post MCAo


Code is provided on a per figure basis.
